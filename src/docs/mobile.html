Installing FinBuddy as an App

üì± Android (Chrome)

Automatic Install Prompt:

  1. 1. Open FinBuddy in Chrome browser
  2. 2. Look for "Install App" banner at bottom of screen
  3. 3. Tap "Install" to add to home screen
  4. 4. App icon appears on home screen
  5. 5. Launch from home screen for app experience

Tip: If you miss the prompt, use the manual installation method below.

Manual Installation:

  1. 1. Open Chrome menu (three dots)
  2. 2. Select "Add to Home screen"
  3. 3. Edit app name if desired
  4. 4. Tap "Add"
  5. 5. Confirm placement on home screen
Features After Installation:
  • ‚Ä¢ Full-screen app experience
  • ‚Ä¢ App appears in recent apps
  • ‚Ä¢ No browser UI clutter
  • ‚Ä¢ Faster launch times

üçé iOS (Safari)

Installation Steps:

  1. 1. Open FinBuddy in Safari
  2. 2. Tap the Share button (square with arrow)
  3. 3. Scroll down and tap "Add to Home Screen"
  4. 4. Edit the app name if desired
  5. 5. Tap "Add" in the top right
  6. 6. App icon appears on home screen

Note: iOS only supports PWA installation through Safari browser.
