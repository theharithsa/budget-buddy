üîÑ App Updates & Cache Management

üì¶ Getting Latest App Updates

FinBuddy regularly receives updates with new features, bug fixes, and improvements. When a new version is released, you may need to clear your cache to ensure you get the latest version.

üöÄ Automatic Update Process

  1. 1. New Release: When FinBuddy releases an update
  2. 2. Background Download: Your browser downloads new files automatically
  3. 3. Cache Conflict: Old cached files may prevent new features from loading
  4. 4. Manual Refresh: You need to clear cache to see updates

Pro Tip: FinBuddy uses service workers for offline functionality, which can cache files aggressively for performance.

‚úÖ How to Clear Cache for Updates

Method 1: Built-in Clear Cache Button
  1. 1. Click your profile picture (top-right corner)
  2. 2. Select "Clear Cache & Refresh" from dropdown
  3. 3. Wait for the success message
  4. 4. App will automatically refresh with latest version
Method 2: Manual Browser Cache Clear
  1. 1. Press Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
  2. 2. Or open Developer Tools (F12) ‚Üí Network tab ‚Üí Disable cache
  3. 3. Refresh the page normally

Clear Cache When You Experience:

  • ‚Ä¢ Missing new features mentioned in release notes
  • ‚Ä¢ Interface looks outdated or different from screenshots
  • ‚Ä¢ JavaScript errors or broken functionality
  • ‚Ä¢ Charts or visualizations not working properly
  • ‚Ä¢ App version number seems incorrect
  • ‚Ä¢ Performance issues or unusual behavior

Signs You Have the Latest Version:

  • ‚Ä¢ All features work as documented
  • ‚Ä¢ App version matches current release (v2.5.5)
  • ‚Ä¢ No browser console errors
  • ‚Ä¢ New features are visible and functional
  • ‚Ä¢ Performance is smooth and responsive
  • ‚Ä¢ Charts and animations work correctly

  5. 5. Restart your device if necessary

  • ‚Ä¢ Insufficient device memory (RAM)

  • ‚Ä¢ Corrupted browser cache

üêõ JavaScript Errors & White Screen

  2. 2. Clear Browser Data: Clear cache, cookies, and local storage
  3. 3. Disable Extensions: Turn off ad blockers and other extensions
  4. 4. Incognito Mode: Test in private/incognito browsing
  5. 5. Different Browser: Try Chrome, Firefox, or Safari
  6. 6. Update Browser: Ensure you're using the latest version
  7. 7. Check Console: Open Developer Tools (F12) for error details
