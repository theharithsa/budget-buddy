🚀 Quick Start

Get FinBuddy running locally in under 5 minutes:

# Clone repository
git clone https://github.com/theharithsa/budget-buddy.git
cd budget-buddy

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:5000

📖 Next Steps: Configure Firebase, set up KautilyaAI, and explore the Metrics Explorer!
