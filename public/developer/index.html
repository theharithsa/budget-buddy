Project Overview

Budget Buddy is a comprehensive personal finance management application built with modern web technologies. This documentation provides detailed technical information for developers working on the codebase.

Frontend: React 18 + TypeScript + Vite
Styling: Tailwind CSS + Radix UI
Backend: Firebase (Auth, Firestore, Storage)
Charts: ApexCharts
AI: Google Gemini 2.0 Flash + OpenAI GPT-4 (Fallback)
PWA: Service Worker + Offline Support
