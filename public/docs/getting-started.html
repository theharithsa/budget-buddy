Step 2: Initial Setup

Budget Setup Wizard

The Budget Setup Wizard helps you create realistic budgets based on your income and spending goals.

Wizard Steps:

  1. 1. Enter your monthly income
  2. 2. Choose budget style (Conservative, Balanced, or Aggressive)
  3. 3. Review AI-generated budget suggestions
  4. 4. Customize amounts for each category
  5. 5. Save your budget configuration

Manual Budget Setup

Prefer to set up budgets manually? You can create category-specific budgets one by one.

Manual Process:

  1. 1. Go to Budget Manager tab
  2. 2. Click "Manual Setup" button
  3. 3. Select a category (Food, Transportation, etc.)
  4. 4. Set monthly spending limit
  5. 5. Repeat for all your categories
